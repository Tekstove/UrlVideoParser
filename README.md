# UrlVideoParser

Allow getting video id from link.

Will extract `z7ODF3A0HhE` from `https://www.youtube.com/watch?v=z7ODF3A0HhE`

# Installation
We support [composer](https://getcomposer.org/)
```
composer require tekstove/url-video-parser
```
